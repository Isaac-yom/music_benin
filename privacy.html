Utilisation

2. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Vous authentifier et maintenir votre session
  • Afficher votre profil public aux autres membres
  • Permettre les fonctionnalités sociales (publications, commentaires, likes, abonnements)
  • Vous envoyer des notifications importantes relatives à votre compte
  • Améliorer la sécurité et prévenir les abus

BéninMusic ne vend jamais vos données à des tiers et n'affiche pas de publicités ciblées.

Sécurité

3. Stockage et sécurité

Infrastructure

  • Base de données : Supabase (PostgreSQL) avec chiffrement au repos et en transit (TLS)
  • Mots de passe : Jamais stockés en clair — hachés avec bcrypt via Supabase Auth
  • Médias : Stockés sur Supabase Storage (AWS S3) avec accès contrôlé
  • Frontend : Servi via Vercel avec HTTPS obligatoire

Accès aux données

Seul l'administrateur du projet (Isaac Yom) a accès à la console Supabase. Les données sont protégées par Row Level Security (RLS) : chaque utilisateur ne peut accéder qu'à ses propres données privées.

Partage

4. Partage des données

Vos données ne sont jamais vendues. Elles peuvent être partagées uniquement avec :

  • Supabase : Hébergeur de la base de données (sous-traitant)
  • Vercel : Hébergeur du frontend (logs techniques uniquement)
  • Autorités légales : En cas d'obligation légale uniquement

Vos publications, commentaires et profil public sont visibles par tous les visiteurs du site, connectés ou non.

Conservation

6. Conservation des données

  • Compte actif : Données conservées tant que le compte est actif
  • Après suppression du compte : Données supprimées sous 30 jours, sauf obligation légale
  • Publications supprimées : Effacées immédiatement de la base de données
  • Logs serveur : Conservés 90 jours par Supabase/Vercel
